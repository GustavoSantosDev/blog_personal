# Introducción a Next.js

Next.js es un framework de React que facilita la creación de aplicaciones web modernas. Con características como renderizado del lado del servidor (SSR) y generación de sitios estáticos, Next.js ofrece un rendimiento excelente y una gran experiencia de desarrollo.

## Ventajas de Next.js

- **Rendimiento**: Optimización automática de imágenes, CSS y JavaScript.
- **SEO**: Renderizado del lado del servidor mejora el SEO.
- **Desarrollo rápido**: Hot reloading y una estructura de proyecto intuitiva.

## Empezando

Para crear un nuevo proyecto con Next.js:

```bash
npx create-next-app@latest my-app
```
